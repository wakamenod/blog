projectile-invalidate-cache

[Emacs]

Where did that file I edited two hours ago go? At times like this, I use consult-recent-file and leverage the power of vertico's fuzzy match to find it. However, the list of consult-recent-files tends to retain a lot of old paths when folders have been moved.

In such cases, using projectile-invalidate-cache might solve the problem. In my environment, it was bound to C-c p i. It seems that executing this command will erase all the old paths.
