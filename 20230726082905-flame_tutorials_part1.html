Flame Tutorials Part 1

[Flutter]

Motivation

I've been interested in the Flutter-based game engine, Flame, for a while now. I plan to work through its tutorial.

Project Setup

Firstly, create an empty Flutter project for these tutorials: 

$ flutter create -e flame_tutorials
$ cd flame_tutorials

In my case, I am using fvm for my Flutter projects, so I also run fvm use. 

$ fvm use

Next, add the essential Flame package: 

$ fvm flutter pub add flame

Run the Project

Open the lib/main.dart file and change the main function as follows. MainApp class should be deleted. 

import 'package:flame/game.dart';
import 'package:flutter/material.dart';

void main() {
  final game = FlameGame();
  runApp(GameWidget(game: game));
}

Also, delete the test/widget_test.dart file. Then launch the emulator and run flutter run. 

$ fvm flutter emulator --launch pixel_xl # The emulator name will vary based on your environment
$ fvm flutter run

Once launched, you'll see a black screen. As per the tutorial, this is the expected behavior.
