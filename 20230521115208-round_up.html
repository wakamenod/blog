Method of rounding up with integers only

[Programming]

Always find it hard to recall instantly, hence the note.

Let's consider the case where both integer A and B are positive integers: 

A / B

If you want to make the result of the division above an integer with rounding up the decimal places, it can be realized as follows: 

(A + (B-1)) / B

This allows for rounding up. (Of course, we're talking about languages/environment where division between integers is truncated, such as Java or C.)

  • If A is divisible by B, B-1 gets truncated so it does not affect the original division result.
  • If A is not divisible by B, adding B-1 makes the number one more than the integer of the original division result.
