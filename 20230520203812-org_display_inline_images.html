org-display-inline-images not working

[Emacs]

So, have you tried M-x org-display-inline-images 18 times and checked the image path 23 times, but the inline images still won't appear? When the link is opened with org-open-at-point, the image can be seen correctly in the new buffer. Why??

This might be the answer:

It turns out that when displaying inline images, you must not include a description in the link! I was creating links with org-insert-link and unconsciously included it. OMG!
